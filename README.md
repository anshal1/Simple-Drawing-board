# A Simple Javascript Based Drawing Board

![App Image](image.png)

## **Main Features**

- Select custom colors
- Pencil tool
- Change the thickness of the pencil tool
- Eraser
- Shapes (Square, Circle)
- Line tool
- Fill tool
- Import image
- Export as image

## **Missing Features**

- [ ] Cannot change the position of circle shape
- [ ] Cannot change the postion of line
